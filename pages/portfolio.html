AICO Redesign

•

About the Project

X

Project Description

The AICO website redesign was a complete overhaul of the company's web presence, modernizing their outdated infrastructure and user experience to support business growth and user needs.

Project Motivation

AICO's existing website was built on legacy ASP technology that was expensive to maintain and modify. The site failed to meet modern user expectations and created operational bottlenecks for internal teams.

•

The Problems Solved

X

Technical Infrastructure

  • Outdated Technology: Legacy ASP-based website was difficult to maintain, with vendors quoting hundreds of thousands for website redesigns due to outdated structure.
  • Backend Integration: Website was tightly coupled with backend systems, creating maintenance issues and limiting flexibility.
  • Scalability: Existing architecture couldn't support growth or new features without becoming too expensive.

User Experience

  • Navigation Issues:  Users struggled to find "collections" quickly, impacting conversion rates.
  • Outdated Design: Visual appearance didn't align with modern standards or brand identity.
  • Mobile Problems: Site wasn't optimized for mobile devices.
  • Poor Information Architecture: Content was poorly organized across different user personas (primarily retailer consumers and in-house stakeholders).

Business Alignment

  • Brand Inconsistency: Website didn't reflect focused brand strategy, limiting social media effectiveness.
  • Marketing Integration: Marketing department could not effectively incorporate new ad campaigns into existing website structure.
  • Data Disconnection: No PIM system integration which prevented access to comprehensive product information for Sales Representatives.
•

My Role

X

Lead Designer and Project Coordinator

I owned the redesign from conception to implementation, handling user research, design execution, and cross-functional collaboration.

Key Responsibilities:

  • Conducted user research across three personas
  • Led stakeholder interviews and requirement gathering
  • Designed UX/UI solutions based on user feedback
  • Collaborated with marketing, development, and executive teams
  • Managed project timeline and deliverables

Team Collaboration

Worked with executives (CEO, President, Sales VP, Operations VP), marketing directors, graphic designers, photographers, sales representatives, and customer relations colleagues.

My Process

X

Zoomed Out: Discover → Define → Develop → Deliver

  • Understanding problems through research
  • Synthesizing insights
  • Creating solutions
  • Implementing and testing

Zoom In: Process Steps

Discovery:

  • User Research: Deployed Notion surveys across three personas (AICO Employees, Sales Representatives, Retailers).
  • Observational Studies: Conducted user testing with existing website to identify workflow issues.
  • Stakeholder Interviews: Gathered requirements from executive team on budget, aesthetics, and functionality.

Definition:

  • Data Analysis: Synthesized research to identify core problems and user needs.
  • Persona Development: Created detailed user personas from research insights.
  • Problem Prioritization: Worked with stakeholders to rank issues by business impact.

Development:

  • Wireframing: Created initial mockups using company brand materials and fonts.
  • Iterative Design: Developed multiple iterations based on stakeholder feedback.
  • Marketing Integration: Incorporated current marketing materials and campaigns.

Delivery:

  • User Testing: Conducted pre-launch testing with persona representatives.
  • Implementation: Built solution using Webflow with strategic backend connections.
  • Launch: Deployed new website and continued iterating based on feedback.

Google HEART Metrics:

  • Beginning: Established baseline for Happiness, Engagement, Adoption, Retention, Task Success
  • Middle: Tracked progress during development to validate decisions
  • End: Measured final impact to demonstrate ROI
•

The Solutions

X

Technical Solutions

Infrastructure Modernization:

  • Webflow Migration: Built external website using Webflow, reducing costs from hundreds of thousands to $100
  • Backend Integration: Maintained necessary backend connections while modernizing front-end
  • Smooth Transition: Users experienced no disruption navigating between new and legacy pages

User Experience Improvements:

  • Navigation Redesign: Streamlined collections discovery with improved information architecture
  • Responsive Design: Implemented mobile-first approach for cross-device compatibility
  • Brand Alignment: Modernized visual design to support social media and brand consistency

Feature Development:

  • PIM Integration: Developed API connections for filtered product information access
  • Marketing Support: Created flexible content management for ongoing campaign updates
  • User Flow Optimization: Redesigned key journeys based on persona needs

Stakeholder Solutions

Internal Users:

  • Executives: Delivered modern, cost-effective solution aligned with brand strategy and budget
  • Sales Reps: Provided easy access to comprehensive product information through PIM integration
  • Marketing Team: Created flexible platform for campaign integration and brand consistency
  • AICO Employees: Simplified internal workflows and improved usability

External Users:

  • Retailers: Improved collection discovery and navigation efficiency
  • Consumers: Enhanced mobile experience and streamlined product access
